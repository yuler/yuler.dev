## 背景, 由于 Docker Hub 在大陆被墙, 为了在

```bash
MIRROR_DOCKER_USERNAME=<your_username>
MIRROR_DOCKER_PASSWORD=<your_password>
MIRROR_DOCKER_REGISTRY=<your_registry>

docker pull basecamp/kamal-proxy:v0.8.2
docker tag basecamp/kamal-proxy:v0.8.2 yuler/kamal-proxy:v0.8.2
docker push $MIRROR_DOCKER_REGISTRY/yuler/kamal-proxy:v0.8.2
```
